## 一、为什么要走 Apple 内购？

| 维度 | Web 直接订阅 | Apple 内购 |
|------|------------|-----------|
| **支付方式** | 信用卡/PayPal | 礼品卡余额 |
| **卡片归属地检测** | Stripe 直接校验 BIN 归属地 | **不存在** —— Apple 代收，Anthropic 看不到卡信息 |
| **中国信用卡** | 大概率被拒 | 无关，你根本不用卡 |
| **账单地址** | 需填支持地区的地址 | 只跟 Apple ID 有关 |
| **被封号后** | 钱打水漂 | 向 Apple 申请退款，成功率高 |
| **风控门面** | Anthropic → Stripe → 你的卡 | Anthropic → Apple → (你的余额) |

**一句话：Apple 是支付中间层，Anthropic 看到的是 Apple 付的钱，不是你付的。卡片归属地判断完全不存在。**

### 2.1 价格对比

### 2.2 省钱的两个渠道

**渠道 1：礼品卡本身打折**

美区 App Store 礼品卡在多个渠道长期有折扣：

| 购买渠道 | 折扣 | 到账 | 风险 |
|---------|------|------|------|
| **亚马逊美区** | 偶尔 85-90 折 | 电子卡，15 分钟 | 极低（官方渠道） |
| **Costco** | 通常 85-90 折 | 电子卡 | 极低 |
| **Target** | 偶尔 85 折 + 红卡再减 5% | 电子卡 | 需美卡支付 |
| **淘宝/闲鱼** | ¥6.2-6.8 / $1 | 电子卡 | **高**（黑卡/盗刷/回收） |
| **充值平台**（如网易等） | ¥6.5-7.0 / $1 | 电子卡 | 中（价差小不如自己买） |

**不建议碰淘宝/闲鱼**：黑卡充值可能导致 Apple ID 被拉黑，得不偿失。

**推荐方式**：注册亚马逊美区账号（不需要美国信用卡，礼品卡余额可买苹果礼品卡），等折扣季囤卡。

**渠道 2：免税州 Apple ID**

美国不同州对数字服务的税率不同：

| 州类型 | 税率 | $20/月的实际支出 |
|--------|------|-----------------|
| 免税州（OR/DE/MT/NH/AK） | 0% | $20.00 |
| 低税州 | 4-6% | $20.80-21.20 |
| 高税州（NY/CA） | 8-9% | $21.60-21.80 |

**把美区 Apple ID 的账单地址设为免税州**（推荐俄勒冈州 Oregon，波特兰随便找个地址），就不收税。

### 2.3 实际省钱测算

假设常规汇率 7.15，你买卡 9 折：

```
按年订阅（App Store 年费 $199.99）：
  正价 × 汇率 = $199.99 × 7.15 ≈ ¥1,430
  买卡成本   = $200 × 0.9 × 汇率买到价

  如果亚马逊 85 折囤卡：$200 × 0.85 × 汇率 ≈ ¥1,215
  如果 Costco 9 折：$200 × 0.9 × 汇率 ≈ ¥1,287

  对比 Web 直接订阅（免税州）$199.99 × 7.15 ≈ ¥1,430

  年省：¥143 ~ ¥215
```

如果你按月订 $20/月，省得更多因为月付本来就更贵。

此外，还有一个**间接省钱**：
- Web 订阅封号 → 钱不退，申诉成功率低
- Apple 内购封号 → Apple 退款政策友好，90 天内基本能退

## 三、完整操作步骤

### 第一步：注册美区 Apple ID

**在 iPhone/iPad 上操作**：

1. 打开 App Store → 点右上角头像 → **退出登录**
2. 点 "创建新 Apple ID"
3. 国家/地区选择 **United States**
4. 邮箱：用 **Gmail / Proton Mail**（不要 QQ/163）
5. 手机号：**可以填 +86**（Apple 允许中国手机号注册美区账号）
6. 付款方式：**选 None**（一开始没有 None 选项，往下看）

关键步骤——让 None 出现：

注册过程中 App Store 会提示你绑卡。此时：
1. **先去 App Store 首页**，找一个免费 App（任何免费 App）→ 点 "获取"
2. 弹出登录提示，选 "创建新 Apple ID"
3. 走完注册流程后，付款方式里会出现 **None**
4. 选 None，然后填写美国账单地址

**账单地址填什么**：

找一个免税州的地址。推荐俄勒冈州：

```
街道: 1234 NW Lovejoy St
城市: Portland
州: Oregon (OR)
邮编: 97209
电话: (503) 555-0199
```

### 第二步：购买 Apple 礼品卡

**推荐亚马逊美区**（不需要美卡）：

1. 注册 amazon.com 账号（不要求美国信用卡）
2. 搜索 "App Store Gift Card"
3. 选 **Email delivery**（电子卡）
4. 金额选 $25 / $50 / $100
5. 支付方式：
   - 如果能办美卡（如 BitGet/PokePay），直接付
   - 如果没有，可以在亚马逊上买 Gift Card 也要支付方式，这是个鸡生蛋问题
   - **解法**：用支付宝在某些支持平台买（见下方）

**实际可行的购买路径**：

| 平台 | 可用支付 | 溢价 | 
|------|---------|------|
| **网易 UU** | 支付宝/微信 | 汇率 + 5-10% |
| **支付宝里搜 "美区App Store"** | 支付宝 | 汇率 + 5-15% |
| **Seagm** | 支付宝/微信 | 汇率 + 10-20% |
| **亚马逊美区** | 需美卡 | 无溢价 + 时有折扣 |

如果你能搞到美卡（PokePay / OneKey / BitGet 卡可能是美卡），在亚马逊买是成本最低的。

### 第三步：充值

1. iPhone/iPad 打开 App Store
2. 确认已登录**美区 Apple ID**
3. 右上角头像 → **兑换礼品卡或代码**
4. 输入礼品卡代码（或手机扫描卡片背面的码）
5. 余额即时到账

### 第四步：订阅 Claude Pro

1. 保持**美区 Apple ID 登录状态**
2. App Store 搜索 "Claude" → 下载 Anthropic 官方 Claude App
3. 打开 Claude App → 登录你的 Claude 账号
4. 设置 → **Upgrade to Pro**
5. 弹出 App Store 订阅确认框，点 **订阅**
6. 扣款从 Apple ID 余额中扣除

## 五、常见问题

**Q: 要不要打苹果客服电话绑卡？**
不需要。用 None 方法就能跳过绑卡，全程不需要信用卡。

**Q: 余额不够时怎么办？**
App Store 会自动尝试续费。如果余额不足，App Store 会发邮件/弹窗提示 "你的订阅即将过期，请充值"。再买一张礼品卡兑换即可。

**Q: 账号被封后订阅怎么办？**
在 iPhone → 设置 → 你的头像 → 订阅 → Claude → 取消订阅，或直接在 App Store → 账户 → 购买记录 → 报告问题申请退款。Apple 对未消费部分的退款政策相对宽松。

**Q: Android 能用吗？**
不能。Apple 内购只能在 Apple 设备上完成。可以用朋友的 iPhone 登录一次你的 Claude 账号订阅，之后在其他平台上继续使用。

**Q: 能不能只用一个月的 Apple 内购，到期后用回 Web 订阅？**
可以。Claude 订阅状态是跨平台同步的——不管从哪里订阅，Pro 状态绑定在你的 Claude 账号上。在 App Store 取消后，到期日再去 Web 重新订阅即可。
