LINE 01 · Recommendation
CLI 怎么选？看你是谁、有多少预算
DECISION
情形 A · 严肃工程 / 长任务
→ Claude Code Max ($100 起)
Opus 4.7 + 1M 上下文 + Cowork 视觉派单。日级别 agentic 重构 / 多文件长链路任务的工业标准。重度用户相比直接 API 节省约 93%。
情形 B · 已是 ChatGPT 订阅
→ Codex CLI（免费随附）
无需额外开支。Pro $100 在 5/31 前限时 10x 用量是性价比窗口。缺点：你被 OpenAI 模型锁定。
情形 C · 国内 / 强成本敏感
→ Qwen Code
真开源 + 真国产 + token 屠价。Qwen3-Coder 480B 在 OpenRouter 还有免费档可白嫖。多协议兼容，可以无痛切回 Claude/GPT。
情形 D · 极致 provider 自由
→ Aider
15B tokens/周处理量、4.1M 安装、39K stars——OSS CLI 的稳态王者。git 集成强到可以把它当版本控制的延伸。
情形 E · 想要 Claude Code 的 UX 但要免费
→ OpenCode
147K stars，星速比 Claude Code 还快 4.5 倍，UI/UX 是这一代 OSS CLI 中最像商用产品的。可以塞 DeepSeek / Qwen / 本地 Ollama。

💡 同线对比要点

价格区间
免费起 → Max $200/mo。OSS 三款（Aider/OpenCode/Qwen Code）软件完全免费，只付 token。
模型上限
Claude Code 独家 Opus 4.7 + 1M 上下文。Codex 独家 GPT-5.5。其他都靠 BYOK。
国内可用性
Claude / Codex 需自备网络方案。Qwen Code 是唯一无障碍的官方国内方案。
隐藏成本
BYOK 看似免费，重度使用一个月 token 账单可能超过 Claude Max $100。先估算再选。

💡 同线对比要点

"Claw"是什么？
月之暗面 Kimi Claw 引爆的命名潮：指能跨 session 持久记忆 + 主动协助的桌面 agent。OpenClaw、QClaw 都是这一波的产物。
编码 vs 通用
Claude Desktop 是唯一把"编码 + 通用对话"做进一个 app 的（嵌 Claude Code）。其余偏纯通用助手。
免费档比较
豆包 / OpenClaw 真免费；ChatGPT Free 美国带广告；Claude / Kimi Claw 必须付订阅才解锁。
隐私 / 数据驻留
OpenClaw 本机跑 = 最私密。其余皆云端推理；国内三家（Kimi / 豆包 / QClaw）数据驻留中国大陆。

编码 vs 通用

⚠ ACU / Credit 陷阱
