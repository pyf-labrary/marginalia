## 本期速览

五条主线，从巨头动作落到你的工具栈：

1. **Anthropic 双响炮（本周最重磅）** — Opus 4.8（5·28）登顶 #1，代理编码 64.3%→69.2%、多数基准超 GPT-5.5 与 Gemini，更诚实、代码缺陷漏过率约 4× 更低；同日 **Claude Code 动态工作流**进入研究预览：Claude 当场写 JS 编排脚本、后台扇出数十~上百并行子代理、对抗式校验迭代到收敛，**单次最多调度 1000 个 agent**（`ultracode` 可自动决策）
2. **编码战场加码** — OpenAI 把 Codex 推成平台（Sites 把工作变可分享网站、注解扩到文档表格幻灯片、六大插件聚合 62 应用，且即将进每个 ChatGPT、已上 AWS Bedrock）；谷歌 Gemini 3.5 Flash 以约 4× 速度、超上代 Pro 的基准杀进编程与代理，已进 Antigravity 2.0
3. **英伟达 Computex** — Vera Rubin 平台量产（AI 工厂引擎）；RTX Spark 超级芯片（Blackwell GPU + 20 核 Grace CPU + 128GB 统一内存）要把 Windows 变成能在本地跑代理的 agentic AI OS，黄仁勋称要「重塑 PC」
4. **计价时代** — GitHub Copilot 6·1 起转按量 AI Credits（1 积分 = $0.01，按 token 计；补全仍免费），Anthropic 6·15 起把 Agent SDK 与 `claude -p` 程序化用量拆进独立额度池；算力账本背后是 Meta×Nebius 五年 $27B、xAI 转租 Anthropic、Anthropic 机密递交 IPO（估值 $9650 亿）
5. **你的工具栈** — 2026 没有单一最优解：复杂多文件重构交 Claude Code（+动态工作流）、后台批处理交 Codex CLI、日常编码交 Cursor、行内补全用 Copilot 免费层，再把部分代理迁到 RTX Spark 这类本地算力

## 工程师本周该做的三件事

1. **搭一套分层工具栈** — 把复杂重构、后台批处理、日常补全分给最合适的代理，别用一个工具硬扛所有任务，也别只押一家
2. **打开你的计价仪表盘** — Copilot 与 Anthropic 都已按量跳表，认真给 token 算账：贵模型只投在中等与困难任务，简单的交给免费补全
3. **试一次本地化** — 把一两个常跑的代理迁到本地算力或更便宜的快模型上，既压低成本，也为数据合规留一条后路

## 本期关键词

> 加速 · 计价 · 编排。英伟达把算力铺到桌面，OpenAI 和谷歌让代理更快更强，而 GitHub 与 Anthropic 同时把账单改成了跳表。巨头在加速，规则在变贵——决定一个工程师价值的，不再是会不会用 AI，而是能不能把这一整套工具，编排成自己的杠杆。

### 制作笔记

W22 沿用 W21 的 v3 Remotion 模板（10 段 + 4 章节 Stinger + ticker + 节拍揭示引擎），scene id 不变、组件不动，每周只换 `script.json` 文案、`data.ts` overlay 与 `beats.ts` 的 BEATS 时刻表：

- **主题**：巨头一周大动作 + 一线工程师该用好哪些工具。章节为 01 MODELS / 02 SILICON / 03 THE METER / 04 YOUR MOVE
- **B-roll 全复用**：本期不生成任何新 B-roll，沿用前几周池（终端编排 / 发光工具箱 / 数据中心 / 芯片脉冲 / 代码瀑布 / 黎明办公室等）直接 Loop，`playbackRate 0.75` / `brightness 0.45`，**零生成成本**
- **BEATS 半自动化**：新写锚点匹配脚本，按每条要点的关键词（如 `Opus 4.8` / `动态工作流` / `RTX Spark` / `Claude Code`）在 `captions.json` 段里游标式取被念到那句的 start 秒，自动产出 BEATS，与配音精确咬合（W21 仍是手填）
- **配音**：MiniMax `speech-2.6-hd × male-qn-jingying` 央视财经主播感，10 段约 2900 字、约 7 分 34 秒
- **总成本**：约 ¥4，仅 MiniMax TTS 按量；B-roll / 图像零生成

源码与流水线：[ai-weekly-2026-06-01 build dir](https://github.com/pyf-labrary/marginalia)（Remotion + MiniMax + ffmpeg；BEATS 锚点匹配脚本本期落地）。
